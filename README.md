# Ejercicio Haz 10

Ejercicio - 6

Inicio: Retornar 1 o 0 según el valor de "a" y "b".

## Ingreso datos
- a
- b
- suma

## Proceso
- Ingresar "a"
- Ingresar "b"
- Sumar "a" más "b" y almacenar en "suma"
- Si "a" es igual a 10 ó "b" es igual a 10 ó "suma" es igual a 10
 - Mostrar 1
- Si no
 - Mostrar 0 Fin si

*Fin*
